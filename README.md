## 🛡️ Security & Anti-Theft Measures
This repository contains a **Proprietary Project**. Several security layers have been implemented to protect intellectual property:
- **Domain Locking**: The website is hard-coded to function only on the official `trojan-techwizard.xyz` domain.
- **UI Hardening**: Right-click, "Save Image," and common developer console shortcuts (F12, Ctrl+U) are blocked on the live site.
- **Selection Prevention**: Text selection and copying are disabled to prevent duplication of problem statements and rules.
- **Ownership Watermarking**: Hidden digital signatures are embedded within the HTML and CSS for authenticity verification.
- **Local Developer Bypass**: All security features automatically deactivate when running on `localhost` for smooth development.
